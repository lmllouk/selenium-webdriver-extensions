Remarks
Because of the limitations of the Selenium the only valid types are: Int64, NullableT, Boolean, Nullable, String, IWebElement and IEnumerable. Selenium returns different types depending if element has been found or not. If there's a match a ReadOnlyCollectionT is returned, but if there are no matches than it will return an empty ReadOnlyCollectionT.
